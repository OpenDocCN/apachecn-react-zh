## 下载

### Docker

```
docker pull apachecn0/apachecn-react-zh
docker run -tid -p <port>:80 apachecn0/apachecn-react-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g apachecn-react-zh
apachecn-react-zh <port>
# 访问 http://localhost:{port} 查看文档
```
